Description
This course provides an overview of website development. Students will learn how the various features, including HTML5, CSS, JavaScript, Ajax, multimedia, HTTP, clients, servers, and databases, function together in today's web environment. The course will provide a web development foundation focusing on content and client-side components, with an overview of the server-side technologies. In addition, software and services that are easily incorporated into a website, for example, maps and content management, will be discussed and frontend frameworks will be introduced.
